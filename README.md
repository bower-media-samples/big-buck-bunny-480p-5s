big-buck-bunny-480p-5s
======================

5 first seconds of Big Buck Bunny (480p)

This is intended to be used as a sample video for testing (ex: `<video>` tag).

Installation
------------

Install using bower:

    bower install big-buck-bunny-480p-5s --save-dev
